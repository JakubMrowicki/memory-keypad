About Memory Keypad

Memory Keypad is a pattern based memory mini-game inspired by the indie game Among Us.

Test your memory skills by playing through four difficulty settings.

How To Play

To play the game simply select your difficulty and then press Start Game

Watch the keypad as the pattern is being played for you then repeat it by pressing on the keypad keys. The game will get progressively more difficult with each turn.

Detailed Tutorial

1. The Keypad

The keypad is the main game area. Try to remember the keypad pattern and repeat it by clicking on the keys. You can also click it to start a game if one is not already in progress.

2. Difficulty Settings

Each difficulty will change the length of the keypad pattern as well as your life count. See table below for more details.

Difficulty Level Pattern Length Max Score Lives
Easy 6 2000 1
Normal 8 3500 1
Hard 10 5400 2
Expert 12 7700 2

3. Stages

This area shows you how you are progressing in the game.

4. Life Counter

This is where you see how many lives you have left. Be very careful if you only have 1 life left, no mistakes can be made from here.

5. Score

This is your current score, it will only wipe when you start a new game so that you can see how you did in your last game even if you didn't beat your high score.

Your score increments exponentially, this means that you gain more score for solving longer patterns.

You can also lose score if you're not careful. If you're playing on Hard or Expert difficulty, you have an extra life which permits a mistake. This will cost you score.

6. Personal Best

This area will show your best score. You can also reset this score by pressing on the button below.
